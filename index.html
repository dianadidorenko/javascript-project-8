For what do we need 'form' tag?

Элемент HTML "form" представляет (собой) раздел документа, содержащий интерактивные элементы управления, которые позволяют пользователю отправлять информацию на веб-сервер.

HTML-элемент "form" используется для встраивания звукового контента в документ. Он может содержать один или более источников аудио, представленных с помощью атрибута src или элемента "source" – браузер выберет один наиболее подходящий
